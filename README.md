# 모노파일 구조물 피로 수명 예측 웹사이트

이 Streamlit 앱은 한국 지도에서 동해, 서해, 남해를 구분하여 지점을 선택하고, 해당 지점의 해상 환경을 고려하여 모노파일 구조물의 예상 피로 수명을 계산합니다.

## 설치 및 실행

1. Python 환경 설정 (venv 권장)
2. 패키지 설치: `pip install -r requirements.txt`
3. 실행: `streamlit run main.py`

## 기능

- 한국 지도 표시 (동해: 파랑, 서해: 초록, 남해: 빨강)
- 지점 클릭으로 선택
- 해역별 해상 환경 데이터 기반 피로 수명 계산
- 결과 표시

## 주의

- 해상 환경 데이터는 가상 값입니다. 실제 적용 시 정확한 데이터를 사용하세요.
- 피로 수명 계산은 단순화된 모델입니다.
